# Data Engineering Pipeline - Deputies Expenses (Portal da Transparência - Brazil-IO)

<b> THIS PROJECT IS UNDER DEVELOPMENT!!! </b>

## Introduction

This is a Data Engineering project base on collecting data provided by [Portal da Transparência](https://portaldatransparencia.gov.br/) in Brazil about deputies expenses. The data is collected and disposed for use from [Brasil IO](https://brasil.io/home/), a non-profit organization which democratizes public data for the community.

It was created under a CI/CD pipeline gathering information from API sources, storing them into S3 buckets and ingesting the files onto Databricks Delta Table environment.
